# boBnox File Organizer

A minimal file organizer that automatically sorts files into categorized folders.

## Features

- Organize files by type (Images, Documents, Videos, Audio, etc.)
- SVG icon button interface
- Automatic log file generation for each run
- Dark theme UI

## Requirements

```bash
pip install cairosvg Pillow
```

## Usage

```bash
python bobnox.py
```

1. Click **Browse** to select a folder
2. Click the icon button to organize files
3. Files will be sorted into categorized subfolders
4. A log file will be saved in the organized directory
